Enhancing Depth and Breadth: The Role of Artificial Intelligence in Transforming Open Source Software Development

Co-authors: Xinyu Li, Keongtae Kim

Brief Description/Abstract: This project explores the impact of GPT-4-powered AI developer tools, particularly GitHub Copilot X (CX), on open-source software (OSS) developers. By employing a difference-in-differences framework, the study investigates how these AI tools influence individual developers' productivity, multitasking abilities, and project management within OSS communities. The project focuses on a 25-week observation window around the release of CX in March 2023, analyzing behavioral changes in development patterns, commit activity, and language usage. The findings indicate that AI tools empower developers to manage multiple repositories, enhance code generation, and expand coding skills, particularly for less active contributors.

Keywords: Generative AI, Open-source Software Communities, Code Autocompletion
